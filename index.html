Dicionário de Tags HTML

<div>

Tag que funciona como um contêiner e define um bloco inteiro de conteúdo. Útil para organizar elementos na página.

<img/>

É usada para incorporar imagens em uma página web, através de um link. Os principais atributos são:

  • src: caminho ou URL para a imagem.
  • alt: texto alternativo para a imagem.
  • width e height: largura e altura da imagem.

<script>

Utilizada para a importação de um documento Javascript para interações do lado cliente com conteúdo dinâmico.

<input>

São utilizadas para o recebimento de dados do usuário, possuindo diferença entre elas como o tipo do atributo que irá receber (type).

<ol>

Cria listas ordenadas (com marcação em ordem numérica).

<ul>

Cria listas não ordenadas (sem marcador numérico).

<li>

Cria elementos (itens) para as listas.

<a> (link)

Cria um link e atribui elementos a ele, como:

  • href: vincula a URL do documento a algo.
  • name: o nome âncora.
  • target: identifica se o link deve ser aberto na mesma janela, em nova aba ou nova janela.
  • rel: define o tipo de relacionamento entre o link e o documento.
  • onclick: define um evento JavaScript para o click no item.

<body>

Representa o corpo do documento HTML, ou seja, a parte principal da página, que engloba vários elementos do conteúdo.

<meta>

Serve para armazenar configurações e/ou informações sobre a página, que geralmente são usados para mecanismos de busca e para o navegador.

<head>

Utilizada para definir o começo e o fim do cabeçalho de um documento HTML, a qual também contém informações gerais sobre o documento, incluindo títulos e links.

<p>

Serve para criar parágrafos de texto.

<strong>

Utilizada para destacar palavras, deixando-as em negrito.
